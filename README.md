# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/67df3e3b-e666-428d-86b5-760f13ff5cdf

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Copy [.env.example](.env.example) to `.env` or `.env.local` and add your Gemini API key
3. Run the app:
   `npm run dev`
4. To verify the project health end to end, run:
   `npm run check`
